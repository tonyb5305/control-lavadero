🚗 Lavadero Hijos de Dios - App de Gestión
Una solución móvil robusta diseñada para digitalizar y optimizar la operación diaria de lavaderos de vehículos. Esta aplicación permite pasar del registro en papel a un control digital preciso, mejorando la organización y el servicio al cliente.

✨ Funciones Principales
📊 Dashboard Operativo
Resumen en Tiempo Real: Visualización inmediata de los ingresos del día y el volumen de vehículos atendidos.

Control de Estados: Clasificación de servicios en "Pendientes", "En Proceso" y "Terminados" para evitar cuellos de botella.

📝 Registro y Control de Servicios
Captura de Datos: Registro rápido de placas, marca y modelo del vehículo.

Catálogo de Servicios: Selección de tipos de lavado (Sencillo, Completo, Motor, etc.).

Asignación de Personal: Control de qué trabajador está a cargo de cada vehículo para facilitar el pago de comisiones o nómina.

💾 Gestión de Datos (Offline)
Modo Sin Conexión: La aplicación funciona totalmente sin internet gracias a su base de datos interna.

Persistencia Segura: Toda la información de servicios y clientes queda guardada localmente de forma permanente.

⚙️ Experiencia de Usuario
Interfaz Adaptativa: Soporte para Modo Oscuro, ideal para largas jornadas de trabajo o diferentes condiciones de luz.

Diseño Intuitivo: Flujos simplificados para que el registro de un vehículo tome menos de 30 segundos.

🛠️ Tecnologías que lo hacen posible
Framework: Flutter (Interfaz multiplataforma fluida).

Base de Datos: SQLite (Almacenamiento local de alto rendimiento).

Lenguaje: Dart.
